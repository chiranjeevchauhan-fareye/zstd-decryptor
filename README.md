# ZSTD Base64 Decryptor

A simple, client-side web application to decrypt ZSTD compressed Base64 strings. It uses the `fzstd` library to run completely in your browser without any backend required.

## How to host on GitHub Pages

1. Go to your repository **Settings** on GitHub.
2. Click on **Pages** in the left sidebar.
3. Under **Build and deployment** -> **Source**, select `Deploy from a branch`.
4. Under **Branch**, select the `main` branch.
5. Click **Save**.
6. Your decryptor will be live at `https://<your-username>.github.io/<repo-name>/` within a few minutes.
